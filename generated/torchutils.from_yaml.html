torchutils.from_yaml¶

torchutils.from_yaml(file)¶

Load configuration from YAML file with include constructor.

To include another file use “!include filename”.

Examples

# file: bar.yaml
- 3.6
- [1, 2, 3]

# file: foo.yaml
a: 1
b:
    - 1.43
    - 543.55
c: !include bar.yaml

kwds = from_yaml("foo.yaml")
