torchutils.infer_parallel_device¶

torchutils.infer_parallel_device(device_ids=None)¶

Decide which device to use for data when given device_ids.

For torch.nn.data_parallel(), if multiple GPUs are used, the data only need to be saved in CPU. If single GPU is used, then data must be moved to the GPU.

Parameters

device_ids (list of int, optional) – gpu id list

Outputs:

  • parallel: True if len(device_ids) > 1

  • device: if single-gpu is used then return the gpu device, else return “cpu”
