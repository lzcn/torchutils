torchutils.loss¶

torchutils.loss.contrastive_loss(im, s, margin=0.1, norm=False, reduction='none')[source]¶

Compute the contrastive loss for two modalities.

loss =

sum{im,k}max(0, margin - sim(im,s) + sim(im,s_k)) + sum{s,k}max(0, margin - sim(im,s) + sim(im_k,s))

Parameters

  • im (array of shape (N, D)) – image features

  • s (array of shape (N, D)) – sentence features

  • margin (float, optional) – margin, by default 0.1

  • norm (bool, optional) – whether to normarlize features

  • reduction (str, optional) –

Returns

loss – contrastive loss between two modalities.

Return type

shape (N, 1)
