torchutils.loss¶

torchutils.loss.contrastive_loss(s_data: torch.Tensor, t_data: torch.Tensor, margin=0.2, mask=None, reduction='mean')[source]¶

Compute the contrastive loss for two domains as follows:

\[\ell(t,s) = \frac{1}{n-1}\left( \sum_{i}^{n}\max(0, m - \text{sim}(t,s) + \text{sim}(t,s_i)) + \sum_{j}^{n}\max(0, m - \text{sim}(t,s) + \text{sim}(f_j,s))\right)\]

where \(t\) and \(s\) are the features of different domains of the same sample and \(f_i\) and \(s_j\) are the features of different samples.

We compute the similarity using cosine similarity.

to support more similarity.

Parameters

  • s_data (torch.Tensor) – array of shape \((N, D)\), source features

  • t_data (torch.Tensor) – array of shape \((N, D)\), target features

  • margin (float, optional) – the margin \(m\). Defaults to 0.1.

  • mask (None, optional) – array of shape \((N)\). mask for samples.

  • reduction (str, optional) –

    Defaults to “mean”. Specifies the reduction to apply to the output:

    • ’none’: no reduction will be applied,

    • ’mean’: the sum of the output will be divided by the number of elements in the output,

    • ’sum’: the output will be summed.

Note

If mask is not None, the output with “none” reduction may contains zeros. Such that simply averaging the output will give a smaller result.

Returns

shape \((N)\) or \((1)\)

Return type

torch.Tensor
