torchutils.ot¶

torchutils.ot.sinkhorn(M: torch.Tensor, a: torch.Tensor = None, b: torch.Tensor = None, v: torch.Tensor = None, gamma: float = 0.01, budget: int = 10, stable: bool = True) → Tuple[torch.Tensor, torch.Tensor, torch.Tensor, torch.Tensor][source]¶

Compute sinkhorn divergence and solve the entropic regularization optimal transport problem.

The optimiazation problem is defined as follows:

\[\begin{split}\begin{aligned} P_{\gamma} &=\underset{P\in \Pi(a,b)}{\text{argmin}}<P,M_{xy}>-\gamma E(P)\\ s.t. &P_{\gamma} 1 = a \\ &P_{\gamma}^\intercal 1= b \\ \end{aligned}\end{split}\]

where \(M_{xy}\) is the cost (distance) matrix over two set of points \(x\sim P_x,y\sim P_y\), \(a,b\) is the distribution of \(x,y\) and \(E(P)\) is the entropic regularization.

The solver iterates over the following two steps:

\[\begin{split}\begin{equation} \left\{\begin{aligned} u&=a / K v \\ v&=b / K^\intercal u \end{aligned}\right. \end{equation}\end{split}\]

where \(P_{\gamma}=\text{diag}(u)K\text{diag}(v)\) and \(K=\exp{(-M_{xy}/\gamma})\)

Parameters

  • M (torch.Tensor) – the cost (distance) matrix \(M_{xy}\).

  • a (torch.Tensor, optional) – Dirichlet distribution over \(x\). Defaults to None (uniform).

  • b (torch.Tensor, optional) – Dirichlet distribution over \(y\). Defaults to None (uniform).

  • v (torch.Tensor, optional) – initial state of \(v\) . Defaults to None.

  • gamma (float, optional) – regularization strength. Defaults to 0.01.

  • budget (int, optional) – number of iterations. Defaults to 10.

  • stable (bool, optional) – whether to use stable version.

Returns

  1. value of \(<P,M_{xy}>\)

  2. optimal \(P_{\gamma}\)

  3. solved \(u\)

  4. solved \(v\)

Return type

Tuple[torch.Tensor, torch.Tensor, torch.Tensor, torch.Tensor]
