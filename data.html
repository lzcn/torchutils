torchutils.data¶

Module torchutils.data defines the data reader class and some utility functions.

Data Readers¶

The instance of DataReader class is callable. It calls the method load() to load data. The key argument has different meanings in different readers. Suppose data is saved in key:value manner. Specially, for data saved in file, the key is the relative path.

The supported data readers are:

  • ImageLMDBReader: A data reader with LMDB backend for image raw data.

  • ImagePILReader: A data reader with PIL backend for image.

  • TensorLMDBReader: A data reader with LMDB backend for numpy.ndarray.

  • TensorPKLReader: A data reader with pickle backend for numpy.ndarray.

Converting Images to LMDB¶

Currently, only one simple method, i.e. create_lmdb() is provided for converting the images. This method creates the LMDB data for images in one folder. And use the path of images as the key for retrieval.

Converting Images to Square¶

A new transform ResizeToSquare for converting images to square in added. The longest side (width or hight) is firstly resized to match the given size. The raito of image is fixed. Then white pixes are padded to get a square image.

torchutils.data.create_lmdb(dst: str, src: Union[str, dict], key='relpath')[source]¶

Convert the image in src to dst in LMDB format

Parameters

  • dst (str) – folder to save lmdb file.

  • src (Union[str, dict]) – folder or dict for (key, file) pairs

  • key (str, optional) –

    Defaults to “relpath”. If src is a directory, it decides which key is used.

    • ”relpath”: use the relative path for key

    • ”filename”: use filename for key

    • ”others”: use fullpath for key

Examples

dst = "/path/to/lmdb"
src = "/path/to/images"
create(dst, src, key="relpath")
It reads images in src folder and creates two files under dst folder:

  • data.mdb

  • lock.mdb

The key of each image is the relative path to src.

torchutils.data.getReader(reader: str = None, path: str = None, data_transform: Union[str, Callable] = None, param: torchutils.param.data_param.DataReaderParam = None) → torchutils.data.DataReader[source]¶

Factory for DataReader.

There are following types of data readers:

  • “ImageLMDB”(ImageLMDBReader): image data with LMDB backend.

  • “ImagePIL”(ImagePILReader): image data with PIL backend.

  • “TensorLMDB”(TensorLMDBReader): tensor data with LMDB backend.

  • “TensorPKL”(TensorPKLReader): tensor data saved with pickle.

Parameters

  • reader (str) – reader type

  • path (str) – data path

  • data_transform (Union[str, Callable], optional) – callable or string to eval. Defaults to None.

  • param ([DataReaderParam], optional) – use ReaderPraram to define. Defaults to None.

Returns

a callable instance of DataReader

Return type

DataReader

Examples

reader = "ImageLMDB"
path = "/path/to/lmdb"
data_transform = '''
[
    transforms.RandomHorizontalFlip(),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]),
]
'''
reader = getReader(reader, path, data_transform)

Within the above example, the data_trainsform is defined in string.

class torchutils.data.ResizeToSquare(size)[source]¶

Resize an image to square with white background padded.

It firstly resize the image with fixed ratio maing the longest side (w or h) meets given size. Then white pixels are padded.

Parameters

size (int) – output image size

Example

trans = ResizeToSquare(291)
im_resized = trans(im)
