torchutils.data¶

This module defines the the data reader and some utility functions.

Data Readers¶

The instance of DataReader class is callable. It calls the method load() to load data. The key argument has different meanings in different readers. The supported data readers are:

  • ImagePILReader: A data reader with PIL backend for image. Data is read directly from path using PIL.

  • TensorLMDBReader: A data reader with LMDB backend for numpy.ndarray. Data is saved in key:value format.

  • ImageLMDBReader: A data reader with LMDB backend for image raw data. Data is saved in key:value format.

  • TensorPKLReader: A data reader with pickle backend for numpy.ndarray. Data is saved in key:value format.

Converting Images to LMDB¶

Currently, only one simple method, i.e. create_lmdb() is provided for converting the images. This method creates the LMDB data for images in one folder. And use the path of images as the key for retrieval.

Converting Images to Square¶

A new transform ResizeToSquare for converting images to square in added. The longest side (width or hight) is firstly resized to match the given size. The raito of image is fixed. Then white pixes are padded to get a square image.

class torchutils.data.ResizeToSquare(size)[source]¶

Resize an image to square with white background padded.

It firstly resize the image with fixed ratio maing the longest side (w or h) meets given size. Then white pixels are padded.

Parameters

size (int) – output image size

Example

im  // PIL.Image
trans = ResizeToSquare(291)
im_resized = trans(im)
